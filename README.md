## 刪除單一使用者繳交紀錄  
-c要刪除的Contest ID  
-u要刪除的User ID  
```  
sudo cmsRemoveSubmissions -c 8 -u S01  
```

## 批量刪除(但是需要一個一個按Y確定)  
```
for i in $(seq -w 1 75); do
  sudo cmsRemoveSubmissions -c 8 -u S$i
done

```
## 可以改用以下每次執行自動輸出Y  
```
yes | for i in $(seq -w 1 75); do
  sudo cmsRemoveSubmissions -c 8 -u S$i
done
```
